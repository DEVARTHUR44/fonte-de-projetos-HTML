oque é apropriedade Display em CSS


O atributo display define o comportamento do elemento em relação aos demais elementos do documento. Originalmente, o código HTML possuia dois tipos de elementos: inline, apresentados no navegador de forma contínua, como conteúdo; e block, apresentdos no navegador como blocos de separação e organização do conteúdo.

Display Caracteristicas
Block Para blocos principais de conteudo, elementos que precisam de sua propria linha
inline m HTML, "inline" refere-se a um tipo de elemento que não começa em uma nova linha e ocupa apenas o espaço necessário para seu conteúdo. Ao contrário dos elementos de bloco, que ocupam toda a largura disponível, os elementos inline fluem dentro do conteúdo, como em um parágrafo.
Inline-block O display: inline-block em CSS faz com que um elemento seja tratado como um bloco, mas ainda mantenha o comportamento de um elemento inline, permitindo que vários elementos sejam exibidos lado a lado e que propriedades como largura e altura sejam aplicadas.
Flex O display: flex no CSS transforma um elemento em um contêiner flexível, permitindo que seus filhos (itens flexíveis) sejam dispostos e alinhados de maneira flexível, tanto horizontal quanto verticalmente. É uma ferramenta poderosa para criar layouts responsivos e organizar elementos de forma dinâmica.
